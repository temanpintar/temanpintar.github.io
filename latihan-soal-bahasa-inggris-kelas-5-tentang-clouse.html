=> If the weather is warm next Sunday, we will go to Kuala Simpang

=> If I have time, I will go to the party with you tonight.

=> If Ameera finishes her task on time, she will accompany Ammar to go to the cinema.

Type 2

If I knew English better, I ... (continue) my study abroad.

=> If I knew English better, I would continue my study abroad.

If we ... (have) the money, we would take a trip to Seoul

=> If we had the money, we would take a trip to Seoul

If you ... (leave) right away, you could be there on time.

=> If you left right away, you could be there on time.

Type 3

If you ... (tell) her the truth, perhaps she wouldn't have been so angry.

=> If you had told her the truth, perhaps she wouldn't have been so angry.

If I ... (feel) better, I would have gone with you to visit our grandparents.

=> If I had felt better, I would have gone with you to visit our grandparents.

=> If Ammar had studied more, he would have gotten a higher mark.

PEMBAHASAN:

Conditional Sentence

if conditional sentence merupakan sebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). Dalam kata lain jika kalimat utama didepan, maka anak kalimatnya berada pada kalimat kedua (belakang) dan sebaliknya.

Ada tiga jenis kalimat if conditional:

1. If Conditional Type 1

If conditional type 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

Contoh:  

If I go to Palembang I will meet Sarah  If we have a yacht, we will sail the seven seas If he have more time, he will learn karate If they tell their father, he will be very angry She will spend a year in the USA if it is easier to get a green card If I live on a lonely island, I will run around naked all day

2. If Conditional Type 2

If conditional type 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

Contoh:  

If I were you, I would forgive him If we had a yacht, we would sail the seven seas If he had more time, he would learn karate If they told their father, he would be very angry She would spend a year in the USA if it were easier to get a green card If I lived on a lonely island, I would run around naked all day

3. If Conditional Type 3

If conditional type 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Rumus/ Pola kalimat:

If + S + had + V3 ( past perfect), S + would have + Verb3

Contoh :  

If Mimi had done her homework, she would have not got a punishment. If we had had a yacht, we would have sailed the seven seas If he had had more time, he would have learnt karate If they had told their father, he would have been very angry She would have spent a year in the USA if it had been easier to get a green card If I had lived on a lonely island, I would have run around naked all day

18. buatlah 10 contoh kalimat if clouse suggestion​


Jawaban:   Conditional sentences dengan menggunakan if-clause bisa digunakan untuk mengungkapkan beberapa hal. Di antaranya adalah untuk pengingat (reminder), memberikan perintah (imperative), memberikan saran (suggestion), menyampaikan sesuatu yang merupakan fakta umum (general truth), atau menunjukkan impian akan masa depan (show a dream).

Berikut adalah penjelasan masing-masing disertai 10 contoh kalimat dan artinya:

If Clause sebagai Reminder (Pengingat)

Conditional sentence ini diungkapkan untuk mengingatkan seseorang akan sesuatu yang harus dilakukan jika ingin mendapatkan hasil tertentu. Ini termasuk conditional sentence type 1 yang menggunakan simple present.

If + Simple Present, Subject + Have/Has to/Must + V1

Contoh:

1. If you want to stay healthy, you must eat organic food. (jika kamu ingin tetap sehat, kamu harus makan makanan organik)

2. If she wants to be slim, she has to go to a fitness center. (jika dia ingin ramping, dia harus ke pusat kebugaran)

3. If they want to meet me, they must come before 3 pm. (jika mereka ingin bertemu saya, mereka harus datang sebelum jam 3 sore)

If Clause sebagai imperative (perintah)

Kalimat pengandaian ini digunakan untuk menyuruh seseorang melakukan sesuatu jika dia/mereka menginginkan terjadinya suatu kondisi.

If + Simple Present, V1 + Object/Adverb

Contoh:

1. If you want to finish your homework, stop playing mobile games now! (Jika ingin menyelesaikan PRmu, berhenti main game HP sekarang juga).

2. If you want to take a good picture, choose a correct setting for the camera! (Jika ingin mengambil gambar yang bagus, pillih pengaturan yang benar untuk kameranya).

3. If you want to win this match, be confident and never surrender! (Jika ingin memenangkan pertandingan ini, percaya dirilah dan jangan pernah menyerah!).

If Clause untuk Memberikan Saran (Suggestion)

Conditional sentence yang satu ini diungkapkan seseorang untuk memberikan saran akan sesuatu yang seharusnya dilakukan jika ingin mendapatkan hasil yang diinginkan.

If + Simple Present, S + Should /Had Better/ Ought to + V1

If + Simple Present, Why don't + S + V1?

Contoh:

1. If you want to get better, you should see a doctor. (Jika ingin cepat sembuh, seharusnya kamu ke dokter).

2. If you want to get up early in the morning, why don't you stop watching TV. (Jika ingin bangun lebih pagi, seharusnya kamu berhenti nonton TV).

3. If you want to speak English well, you had better join a good English course. (Jika ingin lancar bicara bahasa Inggris, seharusnya kamu ikut kursus bahasa Inggris yang bagus).

If Clause untuk mengungkapkan fakta umum (General Truth)

Merupakan kalimat pengandaian untuk menyatakan fakta atau kebenaran yang diterima secara umum.

If + Simple Present, Future

If + Simple Present, Simple Present

1. If you heat water until 100 degrees of celcius, it will boil. (Jika kamu memanaskan air sampai 100 derajat celcius, airnya akan mendidih).

2. If you throw an apple upward, it will fall downward. (Jika kamu melempar apel ke atas, apelnya akan jatuh ke bawah).

3. If the sun rises, the morning comes. (Jika matahari terbit, pagi datang).

If Clause untuk menunjukkan impian (showing a dream)

Merupakan conditional sentence untuk menunjukkan apa yang ingin dilakukan atau didapatkan seseorang di masa depan jika dia memiliki atau berhasil melakukan sesuatu.

If + Simple Present, Future

Contoh:

1. If I have 1 billion rupiahs, I will build a hotel. (Jika punya uang 1 miliar, saya akan membangun hotel).

2. If I graduate from high school, I will continue my study to Japan. (jika lulus SMA, aku akan kuliah di Jepang).

3. If I get the scholarship, I will treat you all. (Jika dapat beasiswa, akan kutraktir kalian semua).

Pengertian Subjunctive

Subjunctive adalah kata kerja yang digunakan untuk membayangkan akibat dari sesuatu yang tidak ada atau tidak terjadi, yang dapat berupa wish (harapan), requirements (kebutuhan), dan suggestion (saran). Atau dengan kata lain, makna kalimat subjunctive ini selalu berlawanan dengan kenyataan atau fakta.

Rumus=
1. Future subjunctive
*Wish=Subject (1) + wish + Subject (2) + Could / Would + Verb 1
2. Present Subjunctive
*Wish=Subject (1) + wish + Subject (2) + Verb 2 / were
*As if /As though=Subject (1) + Verb 1 + as if + Subject (2) + Verb2 / were
*Would rather=Subject (1) + would rather + Subject (2) + Verb 2 / were
*If only=If only + Subject + Verb 2
3. Past Subjunctive
*Wish=Subject (1) + wished + Subject (2) + Had + Verb 3 / been
*As if /As though=Subject (1) + Verb 2 + as if + Had + Verb 3 / been
*Would rather=Subject (1) + would rather + Subject 2 + Had + Verb 3 / been
*If only=If only + Subject + Had + Verb 3 / been


Pengertian Noun Clause

Noun Clause merupakan klausa dependen (klausa yang tidak dapat berdiri sendiri) yang digunakan untuk menggantikan noun (kata benda) dalam suatu kalimat.
Clause sendiri terbagi menjadi beberapa jenis:
1. Noun Clause sebagai subjek kalimat
Rumus :
1. Kata tanya (when, where, who, what, why, how) + S + V + Verb + Object /Kata pelengkap
2. That + S + V + Verb + Object/ Kata Pelengkap
3. If/ Whether + S + V + Verb + Object/ Kata Pelengkap
2. Noun Clause sebagai Objek Kalimat
Rumus Noun Clause:
1. S + V + kata tanya (when, where, why, how, what, why) + S + V
2. S + V + that + S + V
3. S + V + if/ whether + S + V
3. Noun clause sebagai komplemen subjek
Rumus Noun Clouse:
1. S + Linking verb (is, am, are, was, were, be) + kata tanya (who, where, when, how, why, what) + S + V
2. S + Linking Verb (is, am, are, was, were, be) + that + S + V
3. S + Linking Verb (is, am, are, was, were, be) + if/ whether + S + V
4. Noun clause sebagai objek dari preposisi
Rumus Noun Clause:
1. S + V + preposition + kata tanya (who, when, where, how, why, what) + S +V
5. Noun Clause sebagai Adjective Complement
Rumus Noun Clause:
1. S + Verb to be + Adjective + that + S + V
2. S + Verb to be + Adjective + if + S + V
3. S + Verb to be + Adjective + kata tanya (who, why, what, where, when, how) + S + V

Adjective clause adalah klausa yang menjelaskan kata benda. Adjective diletakan mengikuti kata benda yang akan dijelaskan. Adjective clause diawali oleh relative pronoun that (untuk subject/object orang/bukan orang), who (untuk subject orang), whom (untuk object orang), which (untuk subject/object bukan orang), whose (untuk kepunyaan orang),  of which (untuk kepunyaan bukan orang), atau relative adverb where (untuk nama tempat; pengganti there) dan when (untuk waktu; pengganti then).
