13. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5

Jawaban:

No:1. Formulir ujian sekolah/madrasah adalah sebuah jenis formulir yang diberikan kepada setiap peserta ujian tingkat sekolah/madrasah. Formulir ini mulai marak digunakan ketika ujian nasional yang menggunakan metode pilihan ganda dengan menggunakan pensil 2B. Cara ini dinilai lebih baik karena penilaian hasil ujian dapat keluar lebih cepat dan obyektif karena tidak melibatkan penilaian pribadi para guru. Selain itu, soal yang disajikan juga dianggap lebih baik karena dikeluarkan oleh tim pembuat soal yang berasal dari pemerintah pusat.

1. persamaan?

Saya membandingkan formulir tersebut dengan formulir wesel pos. Salah satu persamaan yang terdapat di antara kedua formulir tersebut adalah pencantuman identitas berupa nama.

2. perbedaan?

Untuk menjawab pertanyaan ini, saya membandingkan formulir ujian tersebut dengan formulir wesel. Beberapa perbedaan yang terdapat di antara keduanya mencakup pencantuman tanggal lahir dan nomor identitas.

No:2.

Pada formulir tersebut, saya menemukan beberapa informasi penting yaitu identitas peserta ujian, nomor peserta, tanggal lahir peserta, mata pelajaran yang diujiankan, asal sekolah, serta tanggal pelaksanaan ujian.

No:3. Persamaanya: sama² terdapat identitas hanya saja terdapat nama

Perbedaannya: formulir ujian itu terdapat identitas seperti nama lengkap,kelas, sekolah, matpel,no. Absen,dll.

Kalau formulir pengiriman barang itu terdapat identitas seperti nama pengirim,alamat,nama penerima,dll.

Pembahasan

Soal di atas menanyakan urutan kalimat agar menjadi paragraf prosedur yang tepat. Jika dikaitkan dengan kisi-kisi UN tahun 2017/2018 termasuk ke ruang lingkup materi menulis terbatas level kognitif aplikasi (penerapan). Kompetensi yang diuji yaitu menggabungkan kalimat / mengurutkan kalimat acak.

BACA BEDAH KISI-KISI UN TAHUN2017/2018 BAHASA INDONESIA SMK/MAK

Dalam teks prosedur langkah yang paling mendasar diletakkan pada urutan pertama kemudian diikuti langkah berikutnya yang berhubungan dengan langkah sebelumnya. Langkah yang paling mendasar adalah kalimat nomor (1). Oleh karena itu, urutan kalimat agar menjadi paragraf yang baik pada soal di atas adalah (1), (2), (3), (5), dan (4) atau pilihan jawaban A.

MENGURUTKAN KALIMAT ACAK MENJADI PARAGRAF PADU.

Kalimat acak adalah kalimat lepas yang dapat dijadikan paragraf yang padu dengan cara kalimat utama kemudian diikuti kalimat penjelas yang diurutkan secara logis dan berhubungan satu dengan yang lain.

Dalam teks prosedur langkah yang paling mendasar diletakkan pada urutan pertama kemudian diikuti langkah berikutnya yang berhubungan dengan langkah sebelumnya.

Langkah-langkah mengurutkan kalimat acak menjadi paragraf padu adalah sebagai berikut.

1.      Bacalah dengan cermat.

2.      Pilihlah data yang bersifat umum dan memuat gagasan utama. Gunakan data tersebut sebagai kalimat utama kemudian diikuti kalimat penjelas.

3.      Perhatikan penanda wacana atau kata penghubung seperti pertama, lalu, setelah itu, kemudian, dan akhirnya. Kata pertama biasanya berada di awal teks, sedangkan kata akhirnya berada di akhir teks.

4.      Susun data atau kalimat yang disajikan menjadi paragraf padu
