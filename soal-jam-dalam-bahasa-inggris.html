13. soal nya ada difoto yatolong jawab yaaini jam berapa??? make bahasa inggris#jawab


Answer :

1). 05.30 => It is half past five

2). 11.45 => It is a quarter to twelve

3). 10.15 => It is a quarter past ten

4). 03.10 => It is ten past three

5). 11.35 => It is twenty five to twelve

6). 08.35 => It is twenty five to nine

Pembahasan

Dalam menerjemahkan jam ke bahasa inggris, ada 2 cara

Digital (langsung menerjemahkan sesuai angka nya). Contoh : 05.30 => It is five thirty.Classical (menerjemahkan menit terlebih dahulu baru jam). Contoh nya seperti jawaban di atas.

Berikut ini kosa kata dalam jam menggunakan classical :

O'clock => Tepat (jika jam nya tidak menunjukan lebih/kurang semenit pun)

Past => Lewat (jika jam nya menunjukan menit dimulai dari menit ke 1 sampai 30)

To => Kurang (jika jam nya menunjukan menit ke 31-59, maka yang dihitung adalah kekurangan menit menuju ke jam tersebut)

Half => Setengah/tiga puluh menit

A quarter => Seperempat/lima belas menit.

Contoh :

1.00 => It is one o'clock

1.05 => It is five past one

1.15 => It is a quarter past one

1.45 => It is a quarter to two (karena untuk menuju jam 2, kurang nya sekitar 15 menit lagi. Dan karena menitnya menunjukan lebih dari angka 30 yaitu 45 maka menggunakan "to")

1.30 => It is half past one
