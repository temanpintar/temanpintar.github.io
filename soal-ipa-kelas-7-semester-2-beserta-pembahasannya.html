19. ipa hal 145 kelas 7 semester 2​


Pertanyaan

ipa hal 145 kelas 7 semester 2

Jawaban

Nomer 2 Terdapat Di Gambar

Nomer 3 Terdapat Di Gambar

Nomer 4

Diketahui:

tp = 18.15 ⇔ waktu gelombang primer

ts = 18.19 ⇔ waktu gelombang sekunder

Ditanyakan: Jarak episentrum (Δ) ?

Dijawab:

Dijawab:*Gunakan hukum Laska :

Δ = ((ts - tp)-1) x 1000 km

= (18.19 - 18.15) -1) x 1000 km

= (4-1 dalam menit) x 1000 km

= (3 dalam menit) x 1000 km ⇒ 3 menit = 180 sekon,1 menit=60 sekon

1

= (180 ) x 1000 km

60

= 3.000 km

Jadi jarak ke lokasi permukaan (episentrum) adalah 3.000 km

Untuk Lebih Jelas :

https://brainly.co.id/tugas/9469285

Nomer 5

Diketahui:

Jarak ke episentrum (Δ)=1000 km

Ditanyakan: ts-tp ?

Dijawab:

Dijawab:*Gunakan hukum Laska:

Δ = ((ts - tp)-1) x1000 km ⇒ ts dan tp dalam menit

1000 km = ((ts-tp)-1) x 1000 km

Balikan :

((ts-tp)-1) = 1000 km 

                  1000 km

((ts-tp)-1) = 1

  ts-tp     = 1+1

  ts-tp     = 2 menit

Maka perbedaan waktu gelombang sekunder dan primer adalah sebesar 2 menit atau 2 x 60 sekon = 120 sekon.

MATERI: MICROSOFT OFFICE (WORD)

1. Apa fungsi CTRL + C?

Fungsi CTRL + C (copy) adalah untuk menyalin huruf, file, dsb.

2. Apa fungsi CTRL + V?

Fungsi CTRL + V (paste) adalah untuk menempelkan hal yang sudah dicopy sebelumnya.

3. Apa fungsi CTRL + B?

Fungsi CTRL + B (bold) adalah untuk menebalkan huruf / angka.Contoh: ABC

4. Apa fungsi CTRL + I?

Fungsi CTRL + I (italic) adalah untuk memiringkan huruf / angka.Contoh: ABC

5. Apa fungsi CTRL + U?

Fungsi CTRL + U (underline) adalah untuk memberikan garis bawah dibawah huruf / angka.Contoh: ABC

6. Apa fungsi CTRL + P?

Fungsi CTRL + P menampilkan form untuk print dokumen word. Di CTRL + P juga ada opsi untuk mengatur ukuran kertas yang akan digunakan.

7. Apa fungsi CTRL + N?

Fungsi CTRL + N adalah untuk membuat dokumen baru.

8. Apa fungsi subscript?

Subscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di bawah.Contoh: [tex]X_{2}[/tex]  (angka "2" berada di bawah)

9. Apa fungsi superscript?

Superscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di atas.Contoh: [tex]X^{2}[/tex]  (angka "2" berada di atas)

10. Sebutkan guna Microsoft Word!

Membuat, mengedit dokumen.
