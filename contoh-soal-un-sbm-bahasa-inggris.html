Kelas: XII SMA
Mapel: fisika
Tingkatan: SBMPTN

y = 0,4 sin { 5Π (t+x)}
y = 0,4 sin 5Πt + 5Πx
=>
A = 0,4
w = 5Πt
k = 5Πx

Kerjakan dari nomor 4-2-3-1
(4) panjang gelombang 4 cm
= 2Π/ k
= 2Π/ 5Π
= 2/5
= 0,4 cm {salah}

(2) cepat rambat gelombang bunyi 1 m/ s
V = w/ k
= 5Π/ 5Π
= 1 m/ s { benar}

(3) frekuensi gelombang 2,5 Hz
f = w/ 2Π
= 5Π/ 2Π
= 5/2
= 2,5 Hz { benar}

(1) periode gelombang 0,4 s
T = 1/ f
= 1/ 2,5
= 0,4 s { benar}

Petunjuk!!

B. Jika jawaban 1 dan 3 betul
C. Jika jawaban 2 dan 4 betul

35. soal sbm tolong dijawab cepat nomor 4 yaa.


pencerminan terhadap x = a

............x=a
(s, t) ----------> (2a - s, t)

pencerminan terhadap y = b

.....................y=b
(2a - s, t) ---------------> (2a - s, 2b - t)

garis melalui (s, t) dan (2a - s, 2b - t)

persamaan garis =
(y - y1)/(y2 - y1) = (x - x1)/(x2 - x1)
(y - t)/ (2b - t - t) = (x - s)/(2a - s - s)
(y - t)/(2b - 2t) = (x - s)/(2a - 2s)

garis melalui (0, 0)
(0 - t)/(2b - 2t) = (0 - s)/(2a - 2s)
-t/(2b - 2t) = -s/(2a - 2s)
-t(2a - 2s) = -s(2b - 2t)
-2at + 2st = -2bs + 2st
-2at + 2bs = 2st - 2st
-2at + 2bs = 0
2at = 2bs
a/b = 2s/2t
a/b = s/t
a : b = s : t

= A.
