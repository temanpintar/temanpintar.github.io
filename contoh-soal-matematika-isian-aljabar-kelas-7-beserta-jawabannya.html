Contoh Soal Aljabar :

>> Penjumlahan

1. 3x-3y dan -2y+z

Jawab : 3x + (-3y - 2y) + z

           = 3x - 5y + z

>> Pengurangan

1. -4x³ - x² + 8x dari 2x² - x + 2

Jawab : (2x² - x + 2) - (-4x³ - x² + 8x)

           = 4x³ + (2x² + x²) + (8x - x) + 2

           = 4x³ + 3x² + 7x + 2

>> Perkalian

1. (x² + y)(x + y²)

jawab : (x² × x) + (x² × y²) + (y × x) + (y × y²)

          = x³ + x²y² + xy + y³

          = x³ + y³ + x²y² + xy

>> Pembagian

1. (x² + 3x + 2) ÷ (x + 2)   ------- faktorkan

Jawab : (x + 2) (x + 1) ÷ (x + 2)

          = (x + 1)

Jawaban:

38

Penjelasan dengan langkah-langkah:

diketahui:

a = 1

b = -2

c = 3

kita masukkan setiap nilai a, b, dan c ke dalam operasi aljabar tersebut

3ab² + c - 8b - 3a² + 10

= 3(1)(-2)² + 3 - 8(-2) - 3(1)² + 10

= 12 + 3 - (-16) - 3 + 10

= 12+3+16-3+10

= 38

jadi, nilai dari aljabar tersebut adalah 38
