7. Tuliskan rumus persamaan lingkaran kelas xi smk dan contohnya, buat persentasi besok, syukron


Rumus persamaan lingkaran tergantung kepada

nilai titik pusat ( a,b) dan besarnya jari-jari

lingkaran.

jari - jari lingkaran yang merupakan selisih jarak

antara titik pusat dengan suatu titik ( x,y)

yang merupakan tepi dari lingkaran/ melalui titik

tersebut.

dimisalkan :

suatu lingkaran dengan titik pusat P (a,b) melalui

titik Q ( x,y ) dirumuskan  :

jarak PQ : ( jari-jari )

PQ² = (x - a)²+ (y-b)²

sehingga

r² =  (x - a)²+ (y-b)²


contoh soal 1):

Tentukan persamaan lingkaran yang berpusat

di (2, 3) dan berjari-jari 6!

Jawab :

P (a,b) --> ( 2,3)

r = 6

r² =  (x - a)²+ (y-b)²

6² = (x-2)² + (y-3)²

36 =  (x-2)² + (y-3)²

Jadi persamaan lingkarannya:

(x-2)² + (y-3)² = 36

contoh soal 2):

Tentukan persamaan lingkaran yang

berpusat di (1,2) dan melalui titik (4,-3)!

persamaan lingkaran berpusat di (1,2)

dengan  r adalah :

(x-1)² + (y-2)² = r²

maka untuk mencari r melalui titik (4,-3)

(x-1)² + (y-2)² = r²

(4-1)² + (-3-2)² = r²

(3)² + (-5)² = r²

9 + 25 = r²

34 = r²

jadi persamaamn lngkarannya :

(x-1)² + (y-2)² = 34

Jawaban:

Jawaban buku Bahasa Indonesia kelas XI tugas 7 halaman 38 adalah mengenai penyebab kesalahan dari penulisan kata dalam kalimat.

a. Pewawancara biasanya memberikan kesempatan bagi kita untuk mengajukan pertanyaan diakhir wawancara.

Penyebab kesalahan: Penulisan di- dan akhir seharusnya dipisah.

Perbaikan: Pewawancara biasanya memberikan kesempatan bagi kita untuk mengajukan pertanyaan di akhir wawancara.

b. Di sesi ini biasanya muncul pula pembicaraan mengenai gajih dan tunjangan.

Penyebab kesalahan: Kata gajih bukan merupakan kata baku.

Perbaikan: Di sesi ini biasanya muncul pula pembicaraan mengenai gaji dan tunjangan.

c. Siasati dengan mendahului pekerjaan yang menjadi prioritas hari itu.

Penyebab kesalahan: Akhiran -i pada kata mendahului kurang tepat. Seharusnya menggunakan akhiran -kan sehingga membentuk kata kerja transitif.

Perbaikan: Siasati dengan mendahulukan pekerjaan yang menjadi prioritas hari itu.

d. Ini biasanya karena kita akan berfikir bahwa pekerjaan ini akan memakan banyak waktu dan sulit untuk segera diselesaikan.

Penyebab kesalahan: Kata berfikir bukan merupakan kata baku.

Perbaikan: Ini biasanya karena kita akan berpikir bahwa pekerjaan ini akan memakan banyak waktu dan sulit untuk segera diselesaikan.

e. Ketika tubuh dan otak kerja keras selama beberapa waktu, tentu diperlukan waktu untuk beristirahat sejenak.

Penyebab kesalahan: Kata kerja keras kurang tepat karena karena tidak membentuk kata kerja. Seharusnya bekerja keras.

Perbaikan: Ketika tubuh dan otak bekerja keras selama beberapa waktu, tentu diperlukan waktu untuk beristirahat sejenak.

a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

(x + 2)² + (y - 5)² = r²

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

b. Persamaan lingkaran berpusat (-2,5) menyinggung garis x - 3 = 0 → x = 3

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0

Kalo begitu nanti sore Tiyas mau menengoknya” katanya  dengan penuh semangat.
