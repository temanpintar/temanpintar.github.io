1. ikon Copy ialah ikon untuk menyalin data yg terseleksi (bisa berupa data teks, data gambar, data tabel, dan lainnya).

2. ikon Cut ialah ikon untuk memotong data yg terseleksi (bisa berupa data teks, data gambar, data tabel, dan lainnya).

3. ikon Paste ialah ikon untuk menggandakan data yg sebelumnya sudah disalin atau dipotong.

4. ikon Font Color ialah ikon untuk mengatur warna teks/ huruf.

5. ikon Font Size ialah ikon untuk mengatur ukuran teks/ huruf.

6. ikon Bold ialah ikon untuk mengatur ketebalan teks/ huruf.

7. ikon Italic ialah ikon untuk mengatur kemiringan teks/ huruf.

8. ikon Underline ialah ikon untuk mengatur garis bawah teks/ huruf.

9. ikon New Slide ialah ikon untuk menambahkan slide baru.

10. ikon Layout ialah ikon untuk mengatur tata letak slide yg aktif.

13. apa yang dimaksud dengan microsoft power point


Microsoft Powerpoint adalah sebuah software yang diguankan untuk melakukan presentasi. Microsoft Poweerpoint merupakan salah satu software dari Microsoft Office.

Pembahasan

Microsoft Powerpoint adalah sebuah software yang diguankan untuk melakukan presentasi. Microsoft Poweerpoint merupakan salah satu software dari Microsoft Office. Dalam Microsoft Powerpoint, terdapat berbagai jenis fitur yang dapat digunakan oleh para user. Salah satu fitur dalam Powerpoint adalah template. Template adalah sebuah background yang digunakan untuk memperindah tampilan presentasi.

Dalam Microsoft Powerpoint, lembar kerja dapat disimpan dengan berbagai jenis format, seperti PPT, PPS, POT, dan PPTX. Microsoft Powerpoint memiliki beberapa keunggulan, seperti.

Tampilan yang user friendly yang dapat mempermudah para pengguna.

Tersedia berbagai macam desain background yang digunakan untuk mempercantik tampilan presentasi.

Slide Microsoft Powerpoint dapat diedis sedemikian rupa sesuai keinginan user.

Selain menampilkan text, Microsoft Powerpoint juga dapat disisipkan gambar, tabel, link, dan lain-lain.

26. apa sejarah microsoft power point

Di tahun berikutnya, microsoft powerpoint muncul dengan dukungan warna setelah macintosh berwarna muncul dipasaran.
Pada tanggal 31 Juli 1987, Microsoft mengakuisisi Forethought.Inc dengan powerpoint seharga 14 juta dollar.
