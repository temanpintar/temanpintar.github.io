7. apakah perbedaan Microsoft Word 2007 dengan versi Microsoft Word sebelumnya?


Microsoft word merupakan salah satu aplikasi populer keluaran microsoft. Aplikasi tersebut biasanya digunakan untuk mengolah kata. Sebelum mengeluarkan versi 2007, Microsoft word sebelumnya adalah versi 2003. Perbedaan yang cukup mencolok dari kedua versi ini adalah pada tampilan antar muka atau user interfacenya. Antar muka versi 2007 lebih modern dibandingkan dengan versi 2003.

Microsoft word pertama diterbitkan pada 1983 dengan nama Multi-Tool Word untuk Xenix, Kemudian versi-versi lain dikembangkan untuk berbagai sistem operasi, misalnya DOS (1983), Apple Macintosh (1984), SCO UNIX, OS/2, dan Microsoft Windows (1989). Setelah menjadi bagian dari Microsoft Office System 2003 dan 2007 diberi nama Microsoft Office Word.

19. agar dokumen yang disimpan pada microsoft word 2007 dapat dibuka menggunakan microsoft word 2003 ke bawah maka pada saat menyimpannya pada kolom Save AS Type pilih​


Penjelasan:

Untuk menyimpan dokumen Microsoft Word 2007 agar dapat dibuka menggunakan Microsoft Word 2003 ke bawah, Anda dapat memilih opsi "Word 97-2003 Document" pada kolom "Save As Type".

Berikut adalah langkah-langkahnya:

1. Setelah selesai mengedit dokumen pada Microsoft Word 2007, klik menu "File" di pojok kiri atas.

2. Pilih opsi "Save As" atau "Save As Another File Type". Akan muncul jendela dialog "Save As".

3. Di bagian bawah jendela "Save As", Anda akan melihat kolom "Save As Type" atau "Save as file type" (tergantung pada versi Word yang Anda gunakan). Klik pada kolom tersebut dan pilih opsi "Word 97-2003 Document" atau "Word 97-2003 (*.doc)".

4. Berikan nama file baru dan tentukan lokasi penyimpanan yang diinginkan.

5. Klik tombol "Save" untuk menyimpan dokumen dalam format yang kompatibel dengan Microsoft Word 2003 ke bawah.

Dengan cara ini, dokumen Anda akan disimpan dalam format .doc yang dapat dibuka dan diakses menggunakan Microsoft Word versi lama seperti Word 2003 atau versi sebelumnya.
