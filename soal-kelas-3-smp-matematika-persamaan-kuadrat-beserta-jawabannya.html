(x-2)(x-2) = 7

Cari persamaan kuadrat baru yang akarnya x₁ + 3 dan x₂ + 3​

5±√7

Jawaban:

x² - 10x + 18 = 0

Pembahasan :

Persamaan kuadrat

Bentuk umum persamaan kuadrat adalah ax² + bx + c = 0 , dimana a ≠ 0

Pada persamaan kuadrat memiliki Himpunan Penyelesaian (HP)

Dalam menyelesaikan Himpunan Penyelesaian pada persamaan kuadrat adalah bisa menggunakan cara pemfaktoran, melengkapi kuadrat sempurna, Rumus ABC

Pada umumnya persamaan kuadrat memiliki dua Himpunan Penyelesaian yang berbeda

Penjelasan :

Jika x¹ dan x² akar akar dari persamaan 2x² - 8x - 6 = 0 , Tentukan kuadrat yang akar akar nya [ x¹ + 3 ] dan [x² + 3 ]

2x² - 8x - 6 = 0

a = 2 , b = -8 dan c = -6

x¹ . x² = c/a = -6/2 = -3

x¹ + x² = -b/a = -(-8)/2 = 8/2 = 4

Maka,

(x¹ + 3) + (x² + 3) = (x¹ + x²) + 6 = 4 + 6 = 10

(x¹ + 3)(x² + 3) = x¹(x² + 3) + 3(x² + 3) = (x¹ . x²) + 3x¹ + 3x² + 9 = (x¹. x²) + 3(x¹ + x²) + 9 = -3 + 3(4) + 9 = -3 + 12 + 9 = 18

Maka, persamaan kuadrat baru/ yang lainnya :

x² - (x¹ + x²)x + (x¹ . x²) = 0

x² - 10x + 18 = 0

Kesimpulan :

Jadi, persamaan kuadrat yang akar akarnya (x¹ + 3) dan (x² + 3) adalah x² - 10x + 18 = 0
