5. a. f(x) = 2 (x² - 2x)^(-2/3)

Misal :

u = x² - 2x

u' = du/dx = 2x - 2

f(x) = 2u^(-2/3)

f'(x) = (df/du)(du/dx)

df/du = (-2/3)(2) u^(-2/3 - 1)

df/du = -4/3 u^(-5/3)

f'(x) = (-4/3) {u^(-5/3)} (2x - 2)

f'(x) = (-8/3)(x - 1){(x² - 2x)^(-5/3)}

b. f'(1) = (-8/3)(1 - 1){(1² - 2(1))^(-5/3)}

f'(1) = 0

c. f'(0) = tidak terdefinisi, karena (0² - 2(0))^(-5/3) = tidak terdefinisi atau pembagian dengan nol tidak terdefinisi.

38. tolong bantu matematika peminatan kelas xi


Mapel : Matematika
Bab : Trigonometri

Buktikan !
[tex] \frac{ \cos x . \tan^{2}x }{ \sin x } = \tan x[/tex]
Informasi =
tan x = sin x / cos x
Jadi,
cos x / sin x = cot x atau 1/tan x
[tex]{ \frac{1}{ \tan x} . \tan^{2}x } = \tan x \\ \tan x = \tan x[/tex]
Jadi, cos x . tan^2 x / sin x = tan x
