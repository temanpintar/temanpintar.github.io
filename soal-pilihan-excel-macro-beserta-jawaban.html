1. Tipe dalam penyimpanan yang digunakan untuk menyimpan file yang terdapat macro didalamnya dan macro yang sudah dibuat akan tersimpan di dalam file, yaitu tipe…. Pilih salah satu: a. Excel Binary Workbook b. Excel Macro-Enable Workbook c. Excel Workbook d. Excel Add-In​


Jawaban:

b. Excel Macro-Enable Workbook

Penjelasan:

Berikut ini sedikit penjelasan dalam penggunaannya.

1. Excel Workbook, Menyimpan dengan tipe ini seperti menyimpan file excel atau workbook pada umumnya. Pada file ini tidak terdapat Macro.

2. Excel Macro-Enable Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya. Macro yang sudah dibuat akan tersimpan di dalam file.

3. Excel Binary Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya akan tetapi macro akan tersimpan pada system Microsoft Office Excel.

4. Excel Add-In, Menyimpan dengan tipe ini digunakan untuk menyimpan file macro yang dapat anda gunakan sebagai Add-in atau program tambahan dalam Program Excel.

Langkah-langka Cara menyimpan file Macro :

1. Lakukan Penyimpanan dengan tombol SAVE

2. Akan tampil peringatan “the following features cannot be saved in Macro-free workboks” peringatan ini menginformasikan bahwa file ini terdapat VB Project (macro). Jika anda pilih Yes maka Macro yang sudah dibuat akan dihapus, dikarenakan workbook biasa tidak diperkenenkan untuk menyimpan macro.

3. Pilih No Untuk menyimpan macro.

4. Lakukan penyimpanan didalam folder tujuan. Ubahlah Save as Type Menjadi Excel macro-enable Workbook (xlsm).

5. Klik Save

Dalam kotak nama makro , masukkan nama untuk makro. Membuat nama deskriptif mungkin sehingga Anda bisa dengan cepat menemukan jika Anda membuat makro lebih dari satu.

CATATAN: Karakter pertama dari nama makro harus berupa huruf. Karakter berikutnya bisa berupa karakter huruf, angka, atau garis bawah. Spasi tidak bisa digunakan dalam nama makro; karakter garis bawah berfungsi baik sebagai pemisah kata. Jika Anda menggunakan nama makro yang juga merupakan referensi sel, Anda mungkin mendapatkan pesan kesalahan bahwa nama makro tidak valid.

Untuk menetapkan kombinasi Control (Ctrl) tombol pintasan untuk menjalankan makro, dalam kotak tombol pintasan , ketik huruf apa pun (baik huruf besar atau huruf kecil akan berfungsi) yang ingin Anda gunakan. Sebaiknya gunakan Ctrl + Shift (huruf besar) tombol kombinasi, karena tombol pintasan akan menimpa tombol pintasan Excel default yang sama saat buku kerja yang berisi makro terbuka. Sebagai contoh, jika Anda menggunakan Ctrl + z (Batalkan), Anda akan kehilangan kemampuan untuk Batalkan dalam contoh Excel.

Di daftar Simpan makro di , pilih tempat Anda ingin menyimpan makro.

Secara umum, Anda akan menyimpan makro Anda di lokasi Buku kerja ini , tapi jika Anda ingin makro akan tersedia setiap kali Anda menggunakan Excel, pilih Buku kerja makro pribadi. Saat Anda memilih Buku kerja makro pribadi, Excel membuat buku kerja makro pribadi yang tersembunyi (Personal.xlsb) jika belum ada, dan menyimpan makro di buku kerja ini. Di Windows 10, Windows 7dan Windows Vista, buku kerja ini disimpan dalam C:\Users\user name\AppData\Local\Microsoft\Excel\XLStart folder. Di Microsoft Windows XP, buku kerja ini disimpan di C:\Documents and Settings\user name\Application Data\Microsoft\Excel\XLStart folder. Buku kerja di XLStart folder dibuka secara otomatis setiap kali Excel dimulai, dan kode apa pun yang sudah Anda simpan di buku kerja makro pribadi akan tercantum di dialog makro, yang dijelaskan di bagian berikutnya. Untuk informasi selengkapnya, lihat menyimpan makro.

Dalam kotak Deskripsi , secara opsional Ketik deskripsi singkat tentang apa makro.

Meskipun bidang Deskripsi opsional, sebaiknya Anda masukkan salah satu. Juga, cobalah untuk memasukkan Deskripsi bermakna dengan informasi apa pun yang mungkin berguna untuk Anda atau pengguna lain yang akan menjalankan makro. Jika Anda membuat banyak makro, Deskripsi bisa membantu Anda dengan cepat mengidentifikasi makro yang melakukan apa, jika tidak Anda mungkin harus menebak.
