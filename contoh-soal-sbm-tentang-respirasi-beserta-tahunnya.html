25. tolong dibantu berserta pembahasannya soal sbm no 1 dan 2 ya makasih..


Pusatnya (a,b) karena pusatnya berada pada sumbu y maka a = 0
Jadi P = (0, b)
Melalui Titik potong y = -2x^2 + 8x dan 4x - y = 0 => y = 4x
y = y
4x = -2x^2 + 8x
2x^2 - 4x = 0
2x(x - 2) = 0
x = 0 atau x = 2
x = 0 => y = 4x = 4(0) = 0 => (0,0)
x = 2 => y = 4x = 4(2) = 8 => (2,8)
Persamaan lingkaran pusat (0, b)
x^2 + (y - b)^2 = r^2
Melalui (0,0)
(0)^2 + (0 - b)^2 = r^2
b^2 = r^2
b = r
Melalui (2,8)
2^2 + (8 - b)^2 = r^2
4 + (8 - r)^2 = r^2
4 + 64 - 16r + r^2 = r^2
-16r = -68
r = 68/16 = 17/4 maka b = r = 17/4
Jadi persamaan lingkaran pusat (0, 17/4) dan r = 17/4 adalah
x^2 + (y - 17/4)^2 = (17/4)^2
x^2 + y^2 - 17/2 y + (17/4)^2 = (17/4)^2
x^2 + y^2 - 17/2 y = 0
2x^2 + 2y^2 - 17y = 0

38. Please, no.6,7,8 soal mtk sbm tahun lalu. Jawab salah satunya juga gapapa.


Tinjau F(x) + g(x)
Jika dibagi x²-3x+2 sisanya x+1
Cari akar dari x²-3x+2

x²-3x+2 = 0
(X-2)(x-1) = 0
X = 2 atau x = 1

Untuk x=2
F(2) + g(2) = (2) + 1
F(2) + g(2) = 3

Untuk x=1
F(1) + g(1) = (1)+1
F(1) + g(1) = 2

-------

Tinjau F(x) - g(x)
Jika dibagi x²+x-2 sisanya x
Cari akar dari x²+x-2

x²+x-2 = 0
(X+2)(x-1) = 0
X = -2 atau x = 1

Untuk x=-2
F(-2) - g(-2) = -2

Untuk x=1
F(1) - g(1) = 1

-------
F²(x) + g²(x) jika dibagi x-1 adalah

Cari akar x-1
X-1 = 0
X = 1

F²(1) + g²(1) = ....
Misal:
f(1) = m
g(2) = n

m²+n² = (m+n)² - 2mn
Lihat diatas bahwa f(1)+g(1) = 2,
Maka (m+n)² = 2² = 4

Sekarang cari nilai 2mn.

(m+n)² = m²+n²+2mn
(m-n)² = m² + n² - 2mn
--------------------------------- -
(m+n)²-(m-n)² = 4mn
(2)² - (1)² = 4mn
4-1 = 4mn
3 = 4mn
2mn = 3/2

m²+n² = (m+n)² - 2mn
= 4 - 3/2
= 5/2

Jawabannya A
