16. Pengertian,ciri-ciri,dan contoh soal dari limas belah ketupat?


Pengertian : Belah ketupat merupakan salah satu jenis bangun datar dua dimensi. 

Ciri-ciri :
1. Semua ukuran sisi-sisinya sama panjang.
2. Sudut-sudut yang berhadapan besarnya sama dan di bagi dua sama besar oleh diagonal-diagonalnya,
3. Kedua diagonalnya saling membagi dua sama panjang dan saling tegak lurus satu sama lainnya.
4. Mempunyai dua buah sumbu simetri.
5. Kedua diagonalnya merupakan sumbu simetri dari bangun belah ketupat.
6. Memiliki dua simetri lipat.
7. Memiliki dua buah simetri putar.
8. Jumlah semua sudutnya besarnya adalah 360 derajat.

Contoh soal :
Alas limas berbentuk belahketupat memiliki diagonal 8 cm dan 10 cm. Jika tinggi limas 12 cm, maka volum limas adalah... 
A. 160 cm3 
B. 320 cm3 
C. 480 cm3 
D. 960 cm3 

Pembahasan Contoh Soal
Luas belah ketupat, sebagai alas dari limas:
L = (D1 x D2) / 2
L = (8 x 10) / 2 = 40 cm2

Volume limas
V = (Luas alas x tinggi ) / 3
V = (40 x 12) /3 = 40 x 4 = 160 cm3

25. Tolong bantu saya menjawab pertanyaan ini beserta cara menghitungnya: DIKETAHUI PANJANG SALAH SATU DIAGONAL BELAH KETUPAT 48CM. JIKA KELILING BELAH KETUPAT 100CM MAKA BERAPA LUAS BELAH KETUPAT??


Jawab :

Keliling belah ketupat :
4s = 100
s = 100/4
s = 25 cm

Mencari diagonal kedua,
Pythagoras

a^2 + 24^2 = 25^2
a^2 + 576 = 625
a^2 = 625 - 576
a^2 = 49
a = √49
a = 7 cm
Jadi d2 = 2(7) = 14 cm

Luas belahketupat :
= 1/2 x d1 x d2
= 1/2 x 48 x 14
= 336 cm^2


Semoga membantuMapel: Matematika
Bab: Bangun datar
Pembahasan:

Bangun ruang = Belah ketupat

Diketahui:
diagonal₁ (d₁) = 48 cm
Keliling (K) = 100 cm

Ditanya:
Luas (L) = ?

Cari d₂ terlebih dahulu:

s = K : 4
s = 100 : 4
s = 25

maka:
d₂ = 2 x √(s² - (d₁ : 2)²)
d₂ = 2 x √(25² - (48 : 2)²)
d₂ = 2 x √(25² - 24²)
d₂ = 2 x √(625 - 576)
d₂ = 2 x √49
d₂ = 2 x 7
d₂ = 14

Setelah itu cari Luasnya:
L = 1/2 x d₁ x d₂
L = 1/2 x 48 x 14
L = 336 cm²
