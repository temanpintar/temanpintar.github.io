Bab Perbandingan
Matematika SMP Kelas VII

60 orang x 28 hari = (60 - 12) orang x a
a = 60 x 28 : 48
a = 35 hariPerbandingan berbalik nilai.
Pekerja ----------- Hari
60 ------------ 28
48 ------------ x (60-12 = 48)

60/48 = x/28
5/4 = x/28
4x = (5)(28)
x = (5)(28)/4
x = 5 × 7
x = 35 hari

Berapa hari lebih lama : 35 - 28 = 7 hari.
