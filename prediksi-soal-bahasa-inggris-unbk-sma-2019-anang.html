Jawaban:

11).

Mencari nilai x dari :

[tex]3x + 4 = 2(x - 4) \\ 3x + 4 = 2x - 8 \\ 3x - 2x = - 8 - 4 \\ x = - 12[/tex]

diperoleh nilai x = -12 kita subsitusikan ke :

= x + 6

= -12 + 6

= -6

Jawaban (C) ✅

12).

Diketahui :

panjang (3x - 6) meter

lebar (x + 2) meter

jika keliling taman 56 meter

Ditanyakan :

Panjang dan lebar taman ?

Jawab :

Rumus keliling persegi panjang :

K = 2panjang + 2lebar

56 = 2(3x - 6) + 2(x + 2)

56 = 6x - 12 + 2x + 4

56 = 6x + 2x - 12 + 4

56 = 8x - 8

56 + 8 = 8x

64 = 8x

x = 64/8

x = 8

kita subsitusikan x = 8 ke panjang dan lebar taman :

= panjang taman

= (3x - 6)

= (3.(8) - 6)

= (24 - 6)

= 18 meter

= Lebar Taman

= (x + 2)

= (8 + 2)

= 10 meter

jadi, panjang dan lebar taman sebenarnya adalah

panjang 18 meter dan lebar 10 meter

Jawaban Detail ✅

Jawaban (A)
