12. bagaimana menyelsaikan soal |4x+3|=7


Materi : Persamaan Mutlak

Pembahasan :
|4x + 3| = 7
(4x + 3)² = 7²
(4x + 3)² - 7² = 0
(4x + 3 + 7)(4x + 3 - 7) = 0
(4x + 10)(4x - 4) = 0
x = -5/2
x = 1
