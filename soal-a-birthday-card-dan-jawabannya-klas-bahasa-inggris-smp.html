24. ques klas 8 smpsoal:digmbar​


— Quiz

U4 = 11 = a + 3b

U12 = 35 = a + 11b

Pertama-tama, cari nilai b,

11b + a = 35

3b + a = 11

——————— (-)

8b = 24

b = 3

Cari nilai a,

3b + a = 11

3(3) + a = 11

9 + a = 11

a = 2

Suku ke-20,

= a + 19b

= 2 + 19(3)

= 2 + 57

= 59

Jawab:

C. 59

Penjelasan dengan langkah-langkah:

U4 = 11, U12 = 35.

U20?

Cari beda (b)

b = (U12-U4)/(12-4)

b = (35-11)/(12-4)

b = 24/8

b = 3

Cari suku pertama (a)

b = (U12-a)/(12-1)

U12-a = b(12-1)

a = U12 - b(12-1)

a = 35 - 3(12-1)

a = 35 - 36 + 3

a = 2

Cari U20 dengan rumus

Un = a+(n-1)b

U20 = 2+(20-1)3

U20 = 2+60-3

U20 = 62-3 =

59

Jawaban C. 59
