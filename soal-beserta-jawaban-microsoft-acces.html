1. Query pada microsoft acces


Query adalah semacam kemampuan untuk menampilkan suatu data dari database dimana mengambil dari table-tabel yang ada di database, namun tabel tersebut tidak semua ditampilkan sesuai dengan yang kita inginkan. data apa yang ingin kita tampilkan.

Query adalah suatu extracting data dari suatu database dan menampilkannya untuk “pengolahan” lebih lanjut.Query adalah pertanyaan atau permintaan informasi tertentu dari sebuah basisdata yang ditulis dalam format tertentu.Query adalah perintah-perintah untuk mengakses data pada sistem basis data.Query adalah merupakan bahasa untuk melakukan manipulasi terhadap database, yang telah distandarkan dan lebih dikenal dengan nama Structured Query Language (SQL).

13. Harapan setelah mempelajari Microsoft Acces Adalah?


Manfaat Database

Adapun manfaat untuk kita dalam mempelajari Database adalah sebagai berikut :

Sebagai komponen utama atau penting dalam sistem informasi, karena merupakan dasar dalam menyediakan informasi.

Menentukan kualitas informasi yaitu cepat, akurat, dan relevan, sehingga informasi yang disajikan tidak basi. Informasi dapat dikatakan bernilai bila manfaatnya lebih efektif dibandingkan dengan biaya mendapatkannya.

Mengatasi kerangkapan data (redundancy data).

Menghindari terjadinya inkonsistensi data.

Mengatasi kesulitan dalam mengakses data.

Menyusun format yang standar dari sebuah data.

Penggunaan oleh banyak pemakai (multiple user). Sebuah Database bisa dimanfaatkan sekaligus secara bersama oleh banyak pengguna.

Melakukan perlindungan dan pengamanan data. Setiap data hanya bisa diakses atau di manipulasi oleh pihak yang diberi otoritas dengan memberikan login dan password terhadap masing-masing data.

Agar pemakai mampu menyusun suatu pandangan (view) abstraksi dari data. Hal ini bertujuan menyederhanakan interaksi antara pengguna dengan sistem nya dan Database dapat mempresentasikan pandangan yang berbeda kepada para pengguna, programmer dan administratornya.

30. Jelaskan macam macam field size dalam tipe number pada Microsoft acces


Number adalah salah satu tipe data yang ada pada database Microsoft Access. Number digunakan untuk menyimpan data numeric yang dapat digunakan untuk perhitungan matematis. Number bisa ditambah, dikurang, dibagi, atau pun dikali dengan bilangan lain.

Number pun memiliki beberapa jenis field size yang dapat digunakan sesuai dengan kebutuhan pengguna, jenis field size tersebut antara lain :

byte : untuk menyimpan angka 0 - 255 dan merupakan bilangan bulat memory penyimpanan yang diperlukan 1 byteinteger : menyimpan angka -32768 - 32767 dan hanya berupa bilangan bulat memakai memory sebesar 2 byteslong integer : menyimpan bilangan bulat antara -2147483648 - 2147483647 menggunakan memory penyimpanan 4 bytessingle : menyimpan angka desimal –3.402823E38 sampai  –1.401298E–45 untuk bilangan negatif and from  dan 1.401298E–45 sampai 3.402823E38 untuk bilangan positif dengan memakai memory penyimpanan sebesar 4 bytes, jumlah angka desimal 7double : memakai 8 bytes memory penyimpanan untuk menyimpan bilangan desimal –1.79769313486231E308 sampai –4.94065645841247E–324 untuk bilangan negatif dan 1.79769313486231E308 to 4.94065645841247E–324 untuk bilangan positifdecimal : menyimpan angka desimal dari [tex]-10^{38}-1[/tex] hingga [tex]10^{38}-1[/tex] dengan jumlah desimal 28 memakai memory penyimpanan 12 bytesreplication ID : adalah kumpulan angka hexadecimal unik yang berformat {xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx} di mana tiap x mewakilki satu bilangan hexadecimal. Tipe data ini menggunakan 16 bytes pada penyimpanan memeory. Field ini biasa digunakan sebagai field Autonumber pada saat terjadinya replikasi database.
