8. 172.16.74.56 / 18soal:a. Jumlah Subnetb. Host Per Subnetc. Alamat Subnet markd. Blok subnete. Alamat Host & Broadcast valid1.Subnet2.Host 13.Host Akhir4.Broadcast​


Jawaban:

Dalam notasi IP Address yang diberikan (172.16.74.56 / 18), angka "18" menunjukkan panjang maska subnet dalam bit. Mari kita jawab setiap pertanyaan:

a. Jumlah Subnet:

Jumlah subnet dapat dihitung menggunakan rumus 2^n, di mana n adalah panjang maska subnet dalam bit. Dalam kasus ini, n=18, sehingga jumlah subnet adalah 2^18=262,144 subnet.

b. Host Per Subnet:

Jumlah host per subnet dapat dihitung menggunakan rumus 2^n−2, di mana n adalah panjang maska subnet dalam bit. Dalam kasus ini, n=18, sehingga host per subnet adalah 2^18−2=262,142 host.

c. Alamat Subnet Mark:

Alamat subnet mark adalah alamat IP pertama dalam subnet. Dalam notasi IP, bit host yang terakhir akan diatur ke 0. Jadi, alamat subnet mark adalah 172.16.64.0.

d. Blok Subnet:

Blok subnet adalah kisaran alamat IP yang dapat dialokasikan dalam subnet. Dalam kasus ini, dengan panjang maska 18 bit, blok subnet dimulai dari alamat subnet mark (172.16.64.0) hingga alamat broadcast terakhir dalam subnet. Alamat broadcast adalah alamat dengan semua bit host terakhir diatur ke 1. Jadi, blok subnet adalah 172.16.64.0 hingga 172.16.127.255.

e. Alamat Host & Broadcast Valid:

   Alamat Host Pertama: 172.16.64.1

   Alamat Host Terakhir: 172.16.127.254

   Alamat Broadcast: 172.16.127.255

18. misalnya akan membangun sebuah jaringan dengan ketentuan host yang dibutuhkan antara lain : ruang utama 85 host ruang kedua 100 host ruang server 2 host sedangkan network jaringan adalah 10.10.10.0/24​


Jawaban:

Dalam kasus ini, kita dapat mengalokasikan subnet dari jaringan 10.10.10.0/24 untuk memenuhi kebutuhan host di masing-masing ruangan. Kita perlu menghitung jumlah host yang diperlukan untuk setiap ruangan dan memilih subnet yang memadai untuk setiap ruangan.

1. Ruang Utama Membutuhkan 85 host. Karena kita menggunakan jaringan dengan prefix /24, maka kita memiliki 256 alamat IP yang tersedia. Karena kita perlu 85 host, kita membutuhkan sebuah subnet dengan setidaknya 128 alamat IP (dalam rangka memperhitungkan alamat broadcast dan alamat jaringan). Kita dapat memilih subnet 10.10.10.0/25 atau 255.255.255.128, yang memberikan 128 alamat IP (mulai dari 10.10.10.0 hingga 10.10.10.127) dan memenuhi kebutuhan host yang dibutuhkan.

2. Ruang Kedua Membutuhkan 100 host. Menggunakan subnet 10.10.10.0/24, kita telah menggunakan 256 alamat IP, sehingga kita perlu menggunakan subnet lain untuk ruangan kedua. Kita dapat memilih subnet 10.10.10.128/25 atau 255.255.255.128, yang juga memberikan 128 alamat IP (mulai dari 10.10.10.128 hingga 10.10.10.255) dan memenuhi kebutuhan host yang dibutuhkan.

3. Ruang Server Membutuhkan 2 host. Kita dapat memilih subnet 10.10.10.64/26 atau 255.255.255.192, yang memberikan 64 alamat IP (mulai dari 10.10.10.64 hingga 10.10.10.127) dan memenuhi kebutuhan host yang dibutuhkan. Dalam subnet ini, kita akan memiliki dua alamat IP yang dapat digunakan untuk ruang server (10.10.10.64 dan 10.10.10.65).

Dengan menggunakan subnet di atas, kita dapat membangun jaringan dengan alamat IP dan host yang memenuhi kebutuhan masing-masing ruangan.

26. 1. Tentukan IP Host pertama dan host terakhir 192.168.100.100/272. 202.202.202.176/26Network IP=Broadcast IP=Jumlah Host=Host Pertama=Range IP=Kalau bisa sama caranya ​


Jawaban:

Untuk menentukan IP Host pertama dan Host terakhir dari alamat IP dengan prefix subnet /27 (CIDR), pertama-tama, kita harus menghitung Network IP, Broadcast IP, jumlah host yang tersedia, dan kemudian bisa menentukan Host pertama dan Range IP. Mari kita hitung ### Untuk 192.168.100.100/27:

1. Network IP: 192.168.100.96

2. Broadcast IP: 192.168.100.127

3. Jumlah Host: 32 (2^5, karena ada 5 bit host dalam subnet /27)

4. Host Pertama: 192.168.100.97

5. Range IP: 192.168.100.97 - 192.168.100.126

### Untuk 202.202.202.176/26:

1. Network IP: 202.202.202.128

2. Broadcast IP: 202.202.202.191

3. Jumlah Host: 64 (2^6, karena ada 6 bit host dalam subnet /26)

4. Host Pertama: 202.202.202.129

5. Range IP: 202.202.202.129 - 202.202.202.190

Demikianlah penentuan IP Host pertama dan Host terakhir beserta informasi lainnya untuk kedua alamat IP tersebut.

Fungsi (Invers)

Jika ada suatu fungsi berbentuk [tex]f(x) = \frac{ax + b}{cx + d}[/tex], maka rumus cepatnya:

[tex]\boxed{f^{-1}(x) = \frac{-dx + b}{cx - a} }[/tex]

=========================================

Fungsi f: R → R dinyatakan dengan [tex]f(x) = \frac{x - 3}{2x + 1}, x \neq - \frac{1}{2}[/tex]. Tentukan f⁻¹(4)!

Langkah Penyelesaian:

*Misalkan f(x) = y

[tex]y = \frac{x - 3}{2x + 1}\\\\ y(2x + 1) = x - 3\\\\2xy + y = x - 3\\\\2xy - x=-y-3\\\\x(2y - 1) = -y-3\\\\x = \frac{-y - 3}{2y-1} \\\\\boxed{x = f^{-1}(x) = \frac{- x - 3}{2x - 1} }[/tex]

*Substitusi nilai

Subnetting dilakukan untuk mengelola jaringan dalam sebuah konfigurasi, contohnya di sebuah kantor, untuk membagi jaringan tersebut pada beberapa jalur seperti jalur di beberapa bagian kantor tersebut.
