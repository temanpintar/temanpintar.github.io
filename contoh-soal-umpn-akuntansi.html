2. SOAL UMPN KIMIA POLBAN (UMPN 2019)


Jawaban:

122,5gram

Penjelasan:

pertama buat mula2, reaksi/terurai, sisa

kemudian cari mol KClO3 dan ubah ke gram

Jawaban:

(1) B. 122,5 gram

(2) B. +11,9 kkal

Penjelasan:

(1) [tex]\boxed{\star\ Persamaan\ Reaksi\ \star}[/tex]

[tex]KClO_{3}_{(aq)}+6\ HCl_{(aq)} \rightarrow KCl_{(aq)}+3\ H_{2}O_{(l)}+3\ Cl_{2}_{(g)}[/tex]

◈ Diketahui massa KCl yang terbentuk adalah sebanyak 74,5 gram.

◈ Massa molekul relatif KCl = 39 + 35,5 = 74,5 gram/mol

◈ Mol KCl = [tex]\frac{m}{Mr}[/tex] = [tex]\frac{74,5\ gram}{74,5\ gram/mol}[/tex] = 1 mol

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

[tex]\boxed{\boxed{Untuk\ mencari\ mol\ KClO_{3}, gunakan\ perbandingan\ koefisien\ pada\ reaksi.}}[/tex]

Mol KClO₃ = [tex]\frac{Koefisien\ KClO_3}{Koefisien\ KCl}\times mol\ KCl[/tex]

Mol KClO₃ = [tex]\frac{1}{1}\times 1\ mol[/tex]

Mol KClO₃ = [tex]1\ mol[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Mencari massa KClO₃ yang digunakan: (Mr KClO₃ = 122,5 gram/mol)

Mol KClO₃ = 1 mol

Mol KClO₃ = [tex]\frac{m}{Mr}[/tex]

Massa KClO₃ = [tex]mol \times Mr[/tex]

Massa KClO₃ = [tex]1\ mol \times 122,5\ gram/mol[/tex]

[tex]\boxed{Massa\ KClO_3 = 122,5\ gram}[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Jadi, massa kalium klorat yang diperlukan untuk menghasilkan 74,5 gram kalium klorida sebanyak dalam reaksi ini adalah sebanyak 122,5 gram.

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

(2) Diketahui perubahan entalpi untuk reaksi:

[tex]H_2O_{(l)} \rightarrow H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\ \ \Delta H = +68,3\ kkal\\H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\rightarrow H_2O_{(g)}\ \ \Delta H = -57,8\ kkal\\H_2O_{(l)} \rightarrow H_2O_{(s)}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \Delta H = -1,4\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Perubahan entalpi dari es menjadi uap air:

[tex]H_2O_{(l)} \rightarrow H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\ \ \Delta H = +68,3\ kkal\\H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\rightarrow H_2O_{(g)}\ \ \Delta H = -57,8\ kkal\\H_2O_{(s)} \rightarrow H_2O_{(l)}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \Delta H = +1,4\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Sehingga,

[tex]H_2O_{(s)} \rightarrow H_2O_{(g)} \ \ \ \ \ \ \ \ \Delta H =(68,3-57,8+1,4)kkal[/tex]

[tex]H_2O_{(s)} \rightarrow H_2O_{(g)} \ \ \ \ \ \ \ \ \Delta H = +11,9\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Jadi, perubahan entalpi yang terjadi untuk mengubah es menjadi uap air adalah sebesar + 11,9 kkal.

8. contoh soal akuntansi keuangan​


Jawaban:

Contoh Soal #1:

Pertanyaan:

Sumber daya yang dimiliki suatu perusahaan disebut?

(a). Aset

(b). Kewajiban

(c). Persamaan akuntansi

(d). Ekuitas pemilik

Jawaban:

Jawaban (A) aset adalah sumber daya yang dimiliki oleh perusahaan.

Utang perusahaan disebut kewajiban (jawaban B).

Dan hak pemilik disebut ekuitas pemilik (jawaban D).

Hubungan antara aset, kewajiban, dan ekuitas pemilik dinyatakan sebagai persamaan akuntansi (jawaban C).

Contoh Soal #2:

Pertanyaan:

Daftar aset, kewajiban, dan ekuitas pemilik dari suatu entitas bisnis pada suatu tanggal tertentu disebut:

(a). Neraca

(b). Laporan Laba Rugi

(c). Laporan Ekuitas Pemilik

(d). Laporan Arus Kas

Jawaban:

Jawabannya adalah (a) neraca.

Neraca adalah daftar aset, kewajiban, dan ekuitas pemilik dari perusahaan pada tanggal tetentu.

Laporan Laba Rugi adalah ringkasan pendapatan dan beban perusahaan untuk suatu periode.

Laporan ekuitas pemilik (jawaban C) adalah laporan yang merangkum perubahan dalam ekuitas pemilik untuk perusahaan perseorangan atau persekutuan selama periode tertentu.

Laporan arus kas (jawaban D) merangkum penerimaan dan pembayaran kas untuk suatu periode tertentu.

Contoh Soal #3:

Pertanyaan:

Jika jumlah aset naik Rp 20.000.000 selama periode berjalan dan jumlah kewajiban naik Rp 12.000.000 pada periode yang sama.

Maka jumlah perubahan dalam ekuitas pemilik untuk periode tersebut adalah:

(a). Naik Rp 32.000.000

(b). Turun Rp 32.000.000

(c). Naik Rp Rp 8.000.000

(d). Turun Rp 8.000.000

Jawaban:

Persamaan akuntansi = Kewajiban + Ekuitas Pemilik (Modal)

Jadi, jika aset naik Rp 20.000.000 dan kewajiban naik Rp 12.000.000, ekuitas pemilik harus naik sebesar Rp 8.000.000 (jawaban C), seperti ditunjukkan dalam perhitungan berikut ini:

Rp 20.000 = Rp 12.000 + Ekuitas Pemilik

Rp 20.000 – Rp 12.000 = Ekuitas Pemilik

Ekuitas = Rp 8.000

Contoh Soal #4:

Pertanyaan:

Bila pendapatan sebesar Rp 45.000.000, beban Rp 37.500.000 dan penarikan modal oleh pemilik sebesar Rp 10.000.000.

Maka jumlah laba bersih atau rugi bersih adalah:

(a). Laba bersih Rp 45.000.000

(b). Laba bersih Rp 7.500.000

(c). Rugi bersih Rp 37.500.000

(d). Rugi bersih Rp 2.500.000

Jawaban:

Laba bersih adalah selisih pendapatan atas beban, yaitu Rp 7.500 (jawaban B).

Jika beban lebih besar dari pendapatan, selisihnya adalah rugi bersih.

Penarikan oleh pemilik adalah kebalikan dari investasi pemilik di perusahaan dan tidak mempengaruhi jumlah laba atau rugi besih.

9. soal terlampir. umpn kimia 2019


berarti

jumlah mol Cu = 40/64 = 0,625

jumlah mol S = 4/32 = 0,125

2Cu. + S ---> Cu2S

awal 0,6125 0,125. -

reaksi 0,25. 0,125. 0,25

akhir. 0,375. - 0,125

sisa 0,375 Cu yang gak bereaksi

berat zat yg gak bereaksi =

0,375 x 64 = 24 gram

Jawaban:

(B.) 24 gram

Penjelasan:

Pertama, cari jumlah mol masing-masing zat yang bereaksi.

Mol Cu = [tex]\frac{m}{Ar}[/tex] = [tex]\frac{40\ gram}{64\ sma}[/tex] = 0,625 mol

Mol S = [tex]\frac{m}{Ar}[/tex] = [tex]\frac{4\ gram}{32\ sma}[/tex] = 0,125 mol

⟡ ⟡ ⟡

Kemudian, kita reaksikan kedua zat tersebut.

[tex]\boxed{\bold{Persamaan\ Reaksi}}[/tex]

[tex]\ \ \ \ \ \ 2Cu_{(s)}\ +\ \ \ S_{(s)}\ \ \ \rightarrow\ \ Cu_2S_{(s)}\\m\ \ .625\ mol\ \ \ .125\ mol\ \ \ \ \ \ \ \ \ - \\r\ \ \ .250\ mol\ \ \ .125\ mol\ \ \ \ \ \ .125\ mol\\s\ \ \ .375\ mol\ \ \ \ \ \ \ -\ \ \ \ \ \ \ \ \ \ .125\ mol[/tex]

⟡ ⟡ ⟡

Di akhir reaksi masih terdapat sisa mol Cu sebanyak 0,375 mol. Sehingga massa zat yang tidak bereaksi tersebut adalah:

Massa Cu = mol Cu sisa × Ar

Massa Cu = 0,375 mol × 64 sma

[tex]\boxed{Massa\ Cu=24\ gram}[/tex]

⟡ ⟡ ⟡

Jadi, massa zat tembaga yang tidak bereaksi dengan belerang dalam reaksi tersebut adalah sebanyak 24 gram.
