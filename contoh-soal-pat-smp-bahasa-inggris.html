20. Help me please tolong nich pat smp :/​


Karena total sudut dalam garis lurus adalah 180°, maka :

40° + 7x = 180°

7x = 180° - 40°

7x = 140°

x = 140°/7

x = 20°

Karena garis FG sejajar dengan DE, maka besar sudut FCB sama dengan CBE.

80° + 2y = 7x

80° + 2y = 7(20°)

80° + 2y = 140°

2y = 140° - 80°

2y = 60°

y = 60°/2

y = 30°

Karena total sudut dalam segitiga adalah 180°, maka :

4z + 2y + 40° = 180°

4z + 2(30°) + 40° = 180°

4z + 60° + 40° = 180°

4z + 100° = 180°

4z = 180° - 100°

4z = 80°

z = 80°/4

z = 20°

Jadi, nilai x + y + z = 20° + 30° + 20°

= 70° (A.)
