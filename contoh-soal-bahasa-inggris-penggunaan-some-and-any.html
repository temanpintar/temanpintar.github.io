example: - I don't have any money.

25. jelaskan tentang penggunaan some dan any


some dan any dapat digunakan untuk menunjukkan sejumlah countable atau uncountable nouns. Some untuk kalimat positif, any untuk kalimat negatif dan tanya.

26. pengertian some and any


Jawaban:

Some digunakan dalam kalimat positif. Some bisa digunakan untuk benda countable dan uncountable nouns. Some tidak digunakan pada kalimat negatif atau tanya.

Contoh:

I have some friends.

We bought some coffee.

There are some grapes on the table.

My brothers buy some chairs.

Perlu diketahui bahwa, jika kita menggunakan some untuk benda yang bisa dihitung (countable nouns ), maka benda tersebut haruslah berbentuk jamak/ plural.

ANY

Any dipakai hanya dalam kalimat negatif dan kalimat tanya/interrogative. Dalam kalimat tanya ataupun negatif tidak diperbolehkan memakai some, sama halnya dengan kata any tidak boleh diletakkan dalam kalimat positif. Setelah any, kita bisa meletakkan countable atau uncountable nouns. Bila kata benda tersebut adalah kata benda yang bisa dihitung/countable noun dan lebih dari satu, maka kata benda itupun harus berbentuk jamak/plural.

contoh;

Salah : I don't have any Math book.

Benar : Do they need any knives?

Salah : Do they need any knife?

28. Kapan penggunaan any dan some


Ketika benda, hewan atau manusia menyatakan beberapaingat : some biasa digunakan dalam kalimat positif biasa untuk countable dan uncountable nouns. some tidak digunakan pada kalimat negatif atau tanya.
sementara Any digunakan hanya dalam kalimat negatif dan kalimat tanya. 
e.g :
There are some books on the chair.
There is some salt in the kitchen. 
There aren't any biscuits in jars
Are there any biscuits in jars?

35. ceritakan ke bahasa inggris tentang some dan any serta terdapat conjuction MINIMAL 2 PARAGRAF YA INI TUGAS LES BAHASA INGGRIS PRIVAT AKU KASIH POIN BANYAK NI NO ASAL NO COPAS


Jawaban:

Some; biasanya digunakan ketika kita ingin mengatakan sesuatu yang jumlahnya terbatas, namun kita tidak tahu atau tidak ingin mengatakan jumlah yang kita maksud secara hitungan yang tepat. Kita tidak menggunakan Some pada sesuatu yang umum atau yang kita maksudkan sebagai kategori/ jenis tertentu.

CONTOH;

1.Can you buy some milk? (bisakah kamu memmbelikan saya beberapa susu?) Di sini kita tidak menjelaskan berapa jumlah susu yang kita inginkan untuk dibeli secara jelas, tetapi kita tidak ingin semua susu di seluruh dunia.

2.We need milk to make pancakes (kita membutuhkan susu untuk buat pancakes). Pada contoh ini kita memberi peran kepada susu sebagai kategori umum. Kita berfikir/ beranggapan susu sebagai kategori, tidak memikirkan susu dalam sebuah jumlah. 

CONTOH LAINNYA;

1.Do you have some onions? (apakah kamu memiliki beberapa bawang?)

2.I want to make some chocolate cake for my daughter birthday, do you have some chocolate milk? (aku ingin membuat beberapa kue coklat untuk ulang tahun putriku, apakah kamu punya susu coklat?) 

3.I want some diet coke for tonight when we watching football (aku ingin beberapa kaleng coke untuk mala mini ketika kita nonton bola). 

4.I see some pizza on the table (aku melihat beberapa pizza diatas meja). 

5.I drank some coffee today (aku sudah meminum beberapa gelas kopi hari ini). 

6.She made some cookies yesterday (dia membuat beberapa kue kering kemarin). 

7.He bought some ties two weeks ago (dia membeli beberapa dasi dua minggu lalu). 

-------------------------------------------------------------

Any; pada umumnya, kita menggunakan any sama seperti ketika kita menggunakan some; kita memikirkan jumlah yang tidak kita sebutkan secara spesifik kepada lawan bicara. Yang harus diingat bahwa some dan any digunakan dengan countable plural nouns atau uncountable nouns. Namun, biasanya kita menggunakan any dalam kaliamat tanya atau kalimat negative;

CONTOH;

1.She didn’t buy any tomatoes (Negative sentence) (Dia tidak membeli satu buah tomatpun)

2.She bought some tomatoes (Positive sentence) (dia membeli beberapa tomat)

37. apa perbedaannya some sama any dan penggunaannya untuk apa saja???


kalo some untuk positive tense kalo any untuk negative tenseSome Kata some memiliki arti beberapa dalam bahasa Indonesia. Jadi jika benda yang kita akan sebutkan lebih dari satu kita dapat menggunakan kata some sebagai penjelasnya. Kata some ini kita letakkan persis di depan kata benda yang akan kita jelaskan jumlahnya. Contoh penggunaannya dalam kalimat adalah: I need some apples for making fruit salad. Jangan lupa untuk menambahkan huruf s di belakang kata benda. Hal ini karena dalam bahasa Inggris setiap benda jamak ditambhakan huruf s. Kecuali untuk kata benda yang tidak dapat dihitung seperti water, oil, dan air. Kata some ini hanya dapat digunakan untuk kalimat positif atau kalimat berita saja. Jadi jangan gunakan kata some jika anda akan membuat suatu kalimat negatif atau kalimat tanya.



Any Dalam bahasa Indonesia any memiliki arti banyak. Namun kata any hanya dapat digunakan dalam kalimat negatif atau kalimat tanya saja. Kata any seperti halnya kata some, juga diletakkan persis di depan kata benda. Contoh kalimatnya adalah: I don’t have any foods to be eaten.Do you have any coffee?Pada kalimat pertama adalah contoh any yang digunakan dalam kalimat negatif. Sedangkan kalimat kedua adalah kata any yang digunakan dalam kalimat tanya. Jangan lupa juga untuk menambahkan huruf s pada setiap kata benda yang dapat dihitung.

40. Apa Perbedaan "Some " Dan "Any"dalam penggunaan kalimat,serta berikan contohnya!​


Jawaban:

Some digunakan untuk benda yg jumlahnya tidak dapat dihitung

Any digunakan untuk benda yg jumlahnya dapat dihitung

Jawaban:

Some dan any adalah quantifier untuk menyatakan benda yang dapat dihitung (countable nouns) dan benda yang tak dapat dihitung (uncountable nouns). “Some” digunakan dalam kalimat positif. Contoh: I have some cookies in my bag. “Any” digunakan dalam kalimat negatif dan kalimat tanya. Contoh:Is There any cookies in my bag.
