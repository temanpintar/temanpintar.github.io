a.nyatakan biaya tersebut sebagai persamaan kursi yang diproduksi ,dengan mengasumsikan hubungan antara biaya dan banyak kursi adalah linear.
kemudian gambarkan.

b.tentukan gradiennya,dan jelaskan arti bilangan itu.
c.dari sketsa ,jelaskan makna grafik tersebut.

JAWABAN

Bentuk umum ⇨ y = mx + c
y = biaya produksi
x = jumlah kursi
m = gradien
c = konstanta

Masukkan nilai2 x dan y ke persamaan umum untuk mencari m dan c.
◆Untuk x = 100 maka y = 22.000.000
⇨ 100m + c = 22.000.000 ....... (pers. 1)
◆Untuk x = 300 maka y = 48.000.000
⇨ 300m + c = 48.000.000 ....... (pers. 2)

Eliminasi (1) dan (2) diperoleh
⇨300m + c = 48.000.000
⇨100m + c = 22.000.000
-------------------------------------- -
⇨200m = 26.000.000
⇨m = 26.000.000/200
⇨m = 130.000

Substitusi m = 130.000 ke persamaan (1), maka
⇨ 100(130.000) + c = 22.000.000
⇨ c = 22.000.000 - 13.000.000
⇨ c = 9.000.000

a. persamaan biaya produksi kursi
⇨ y = mx + c
⇨ y = 130.000x + 9.000.000

b. gradien = m = 130.000
Nilai 130.000 ini berarti setiap penambahan satu kursi maka biaya produksi akan bertambah sebesar Rp 130.000,00.

______________
