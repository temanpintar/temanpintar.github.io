30. Soal Latihan Standar UN Detik Detik UN Fisika SMA


A1 = 4. 8 = 32

(x1,y1) = (2,4)


A2 = 4. 4 = 16

(x2,y2) = (6,2)


Koordinat titik berat x

x = ( A1. x1 + A2. x2 )/ (A1 + A2)

x = ( 32. 2 + 16. 6 )/(32 + 16)

x = 160/48 = 10/3


Koordinat titik berat

y = ( A1. y1 + A2. y2 )/ (A1 + A2)

y = ( 32. 4 + 16. 2 )/(32 + 16)

y = 160/48 = 10/3


Koordinat titik berat di (10/3, 10/3)

39. Soal Latihan Standar UN Detik Detik UN Fisika SMA


Hk.Kekekalan Momentum

m₁.v₁ + m₂.v₂ = ( m₁ + m₂ ).V

4. 5 + 1. 0 = ( 4+1 ). V

V = 20/5 = 4 m/s

Kecepatan setelah tumbukan 4 m/s

》tumbukan《

kelestarian momentum

ma va + mb vb = {ma + mb} v'

4 • 5 + 1 • 0 = {4 + 1} v'

20 = 5 v'

v' = 4 m/s {ke kanan}
