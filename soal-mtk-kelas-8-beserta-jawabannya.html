[tex]\sf \dfrac{y-y_1}{y_2-y1} = \dfrac{x- x_1}{x_2 - x_1}[/tex]

[tex]\sf \dfrac{y-3}{6- 3} = \dfrac{x-2}{5- 2}[/tex]

[tex]\sf \dfrac{y-3}{3} = \dfrac{x-2}{3}[/tex]
y - 3 = x - 2
y = x+ 1 atau  x - y + 1= 0

[tex]\sf \dfrac{y-y_1}{y_2-y1} = \dfrac{x- x_1}{x_2 - x_1}\\[/tex]

[tex]\sf \dfrac{y + 5}{-6 + 5} = \dfrac{x+2}{5+2}[/tex]

[tex]\sf \dfrac{y + 5}{-1} = \dfrac{x+2}{7}[/tex]

7(y + 5) = - (x + 2)
7y + 35 = -x - 2
7y = - x - 37 atau  x + 7y +  37= 0

3.  PGL melalui (0,0)  dan  (-6, 3)
[tex]\sf \dfrac{y-y_1}{y_2-y1} = \dfrac{x- x_1}{x_2 - x_1}\\[/tex]

[tex]\sf \dfrac{y-0}{3-0} = \dfrac{x- 0}{-6- 0}\\[/tex]

Dimisalkan :

x = sudut pusat (dalam derajat) yang menbentuk busur dan juring tersebut

r = jari-jari lingkaran

[tex]\\[/tex]

Diketahui : panjang busur = 2,5 cm

[tex]\color{darkred}\sf Panjang~busur=\frac{x}{360}\times 2\times \pi\times r[/tex]

[tex]\sf 2,5=\frac{x}{360}\times 2\times \pi\times r[/tex]

[tex]\sf 2,5\times 360=~x~.~2~.~\pi~.~r[/tex]

[tex]\sf 900=~x~.~2~.~\pi~.~r[/tex]

[tex]\sf 450=~x~.~\pi~.~r[/tex]

[tex]\purple{\sf x=\frac{450}{\pi~.~r}}[/tex][tex]~...~(~i~)[/tex]

[tex]\\[/tex]

Diketahui : luas juring = 5 cm²

[tex]\color{darkred}\sf Luas~juring=\frac{x}{360}\times \pi\times r^2[/tex]

[tex]\sf 5=\frac{x}{360}\times \pi\times r^2[/tex]

[tex]\sf 5\times 360=~x~.~\pi~.~r^2[/tex]

[tex]\sf 1.800=~x~.~\pi~.~r^2[/tex]

Subtitusikan nilai x pada [tex](~i~)~:[/tex]

[tex]\sf 1.800=~\frac{450}{\pi~.~r}~.~\pi~.~r^2[/tex]

[tex]\sf 1.800=~450~.~r[/tex]

[tex]\purple{\sf r~=~4~cm}[/tex]

Sehingga :

[tex]\color{darkred}\sf Keliling~lingkaran~=~2\times \pi\times r[/tex]

[tex]\sf Keliling~lingkaran~=~2\times \pi\times 4[/tex]

[tex]\pink{\sf Keliling~lingkaran}=\pink{\huge{\sf 8\pi}}[/tex]

[tex]\to~(~\red{\bold{A}}~)[/tex]
