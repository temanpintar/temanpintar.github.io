26. I... thirstylast night​


I was thirsty last night

Jawabannya adalah WAS ✔️

PEMBAHASANPast Tense

Sebuah tenses yg digunakan utk menunjukkan suatu Kegiatan atau Peristiwa yg terjadi di masa lampau. Cirinya:

YesterdayLast weekLast night ✔️Last Year

Rumus Past Tense

1. Verbal Sentence

➡️ Kalimat yg berisi kata kerja di dalamnya

(+) S + Verb 2 + Object (-) S + did not + Verb 1 + Object (?) Did + S + Verb 1 + Object?

2. Nominal Sentence

➡️ Kalimat yg tak berisi kata kerja di dalamnya.

(+) S + Tobe (was/were) + Adjective (-) Tobe (was/were) + not + Adjective (?) Tobe (was/were) + S + Adjective ?

KESIMPULAN

(+) S + To Be (was/were) + Adjective

➡️ I + was + thirsty + last night

➡️ Saya kehausan semalam/tadi malam

Kalimat tersebut adalah kalimat bentuk NOMINAL SENTENCE diketahui melalui tidak adanya kata kerja dan setelah titik2 adalah Adjective “thirsty”karena ada LAST NIGHT (semalam/tadi malam) maka kalimat bentuk lampau yg berarti PAST TENSE
